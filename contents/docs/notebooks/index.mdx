Notebooks combine PostHog's tools into a single space for analysis. Instead of jumping between dashboards, insights, replays, and experiment results, your entire analysis chain lives on one page.

Notebooks store content as markdown, making them human-readable and easy to version control. This also means the content you see is exactly what PostHog AI reads and writes when you collaborate with it.

Use notebooks to organize ad-hoc analysis, bug investigations, internal documentation, and feature releases – solo or with your team.

To get started, go to the [**Notebooks** tab](https://app.posthog.com/notebooks) and click **New notebook**.

## Adding content to notebooks

To start adding content to notebooks, start typing. There are three main ways to add content to notebooks:

1. **Slash commands.** Within a notebook, type `/` and a menu will pop up with options for you. For example, `/trend` creates a trend insight.

## Notebook content types

### Text

Notebooks supports the standard text editing features via Markdown. These include:

- \# Heading 1
- \## Heading 2
- \### Heading 3
- \- List
- `1.` Numbered list
- `- [ ]` Task list (unchecked)
- `- [x]` Task list (checked)
- \**bold**
- \_italic\_
- \`code\`

### Insights

You can add all of PostHog's insight types natively in Notebooks. These include trends, funnels, retention, paths, stickiness, lifecycle, and SQL.

Each of them can be completely customized within notebooks like you would in insights. To do this, click the three-line reverse pyramid icon.

### Images

You can add both images (.png, .jpg, .jpeg, .bmp) and GIFs to your notebooks using the slash command or by dragging and dropping. 

### Links

You can create preview links for areas within PostHog, like dashboards and notebooks, by dragging and dropping them into notebooks.

You can create external links by highlighting text and then pasting the link in the box.

### LaTeX

You can include mathematical formulas and equations in your notebooks using LaTeX notation. This is perfect for data analysis, statistical explanations, or any mathematical documentation you need to include.

You can add LaTeX content in two ways:

1. Using the slash command and selecting **LaTeX** to create a dedicated math block.
2. Inline by wrapping your formula with double dollar signs (`$$`). For example, typing `$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$` will render as a formatted equation.


### Tables

You can add tables to your notebooks for organizing data.

To add a table:

- Use the slash command `/table` or click the **+** icon and select **Table**
- Paste tabular data directly from Excel, Google Sheets, or any CSV/TSV source

When you click inside a table, a menu appears to add or delete rows and columns.

### Inline AI chat

Type `/ai` in a notebook to start a conversation with [PostHog AI](/docs/posthog-ai). This creates a persistent chat block where the AI can answer questions, generate insights, and insert visualizations directly at that location.

Ask it to analyze a trend, explain an anomaly, or create a funnel based on your description. Any visualizations or insights it generates appear inline, and you can continue to edit and refine them.

## Collaboration

Notebooks support Google Docs-style comment threads so your team can discuss specific parts of an analysis.

### Inline comments

Highlight text and click the comment icon in the formatting toolbar to start a thread. The highlighted text turns yellow, and a comment card appears in the right gutter (or inline on narrow screens). Click a highlight to jump to its thread.

Team members can reply, and all replies stay with the notebook. Delete a thread by clicking the trash icon – this removes the highlight but keeps the underlying text.

### Block comments

Any block – queries, tables, code, images – can have its own discussion. Hover over a block and click the comment icon in the left gutter to attach a thread.

### Mentions

Type `@` followed by a teammate's name to mention them. They'll get a notification and can click through to the thread directly.

## Canvas

A canvas is like a Notebook whose entire contents is persisted in the URL so that it can be easily shared with members of your organisation. Canvases support all the same content types as Notebooks but do not get persisted to the database or come with any form of history tracking. They are a great way of doing some quick explorations that you may want to share with a colleague but don't necessarily need to save for future use. The shareable link changes as you make edits, so updates to you notebook will require a new link to be shared.

## Frequently asked questions

### How much do notebooks cost?

Notebooks are free. [Sign up for free](https://app.posthog.com/signup) to try them out.

### What are the differences between dashboards and notebooks?

Notebooks incorporate all the components of PostHog. Dashboards specifically focus on analytics, insights, and events. 

Dashboards are great for metrics you want to check repeatedly and understanding the status of your product. Notebooks are great for doing analysis and piecing data together into a narrative.

### Can I share a notebook?

Yes, members of your organization can access and edit the notebooks you create. You can click the share icon at the top of your notebook to get a link to provide them direct access.

Sharing externally isn't available yet, but you can request it in the notebook sharing menu.

### Do notebooks support real-time multiplayer editing?

Yes. Your changes autosave, and you see updates from other editors in real time. If two people edit at the same time, PostHog uses a three-way merge to combine changes. When the merge can't resolve cleanly, a conflict dialog shows exactly what changed so you can choose which version to keep or copy text from the other version.

### How do I upgrade an older notebook to markdown format?

Notebooks created before the markdown update use a legacy rich-text format. When you open one of these notebooks, an upgrade dialog appears with the option to convert it. The upgrade is optional – your original notebook stays intact until you convert it.
